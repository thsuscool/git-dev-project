준비하기
SGIS 지도 API는 인증키 발급을 받아야 사용할 수 있습니다. 사용자가 발급받은 키로 트래픽을 관리하여 모든 사용자에게 원활한 서비스를 제공하기 위함입니다. 열쇠모양의 버튼 을 클릭하면 인증키 생성 관리 페이지 로 이동합니다. 아직 발급받은 키가 없다면 해당 페이지에 접속하여 키를 발급 받으세요.

테스트키 신청방법

약관동의 → 정보입력 → 테스트키 신청 → 즉시발급

사용키 신청방법

마이페이지 → 테스트키정보수정 → 신청 → 승인 → 사용

마커생성
마커는 지도위에 특정 지점을 표시하기 위한 요소입니다.
기본적으로 마커를 생성하기 위해서는 생성될 위치가 지정되어야 합니다
